Key Concepts
RBAC vs. Access Policies
Azure RBAC (recommended) integrates with Entra ID using standard role assignments. The legacy Key Vault access policies model is being retired вҖ” always choose RBAC for new deployments.
Groups Over Individuals
Assign roles to Entra ID groups, not users. Onboarding and offboarding becomes a group membership change вҖ” one update propagates to every resource the group has access to.
Static Private IP
Dynamic IPs change on restart. Set static when NSG rules, DNS entries, or monitoring configs reference the IP directly вҖ” a dynamic change breaks those dependencies silently.
Disk Init in Windows
Azure data disks arrive uninitialized. Azure handles provisioning the disk to the VM; Windows Disk Management handles the filesystem вҖ” identical to initializing a new SAN LUN on any Windows Server.
Policy Initiatives
An initiative bundles multiple policies into a single assignable object. Update the initiative once вҖ” all assignments inherit the change without individual reassignment.
Budget Alerts вү  Spending Limits
Alerts notify you вҖ” they don't stop charges. Azure won't shut off resources at the threshold. Use alerts as triggers for manual review or automated runbook actions.
